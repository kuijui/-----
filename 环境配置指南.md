# 环境配置指南

本文档指导你从零配置运行环境，预计耗时 **30-60 分钟**。

---

## 第一步：安装基础软件

| 软件 | 版本要求 | 下载地址 |
|------|----------|----------|
| Node.js | >= 18.0 | https://nodejs.org |
| MySQL | >= 8.0 | https://dev.mysql.com/downloads/mysql/ |
| Redis | >= 7.0 | https://github.com/tporadowski/redis/releases（Windows版）|
| 微信开发者工具 | 最新稳定版 | https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html |

> **检查是否已安装**（在终端运行）：
> ```bash
> node -v       # 应显示 v18.x.x 或以上
> mysql --version
> redis-cli ping  # 应返回 PONG
> ```

---

## 第二步：获取必要账号和密钥

### 2.1 微信小程序 AppID（必须）

1. 访问 [微信公众平台](https://mp.weixin.qq.com)
2. 注册/登录 → 选择「小程序」
3. 「开发」→「开发管理」→「开发设置」
4. 复制 **AppID** 和 **AppSecret**

### 2.2 AI 模型 API Key（二选一即可）

**方案A - OpenAI（国际版，需翻墙）**
1. 访问 https://platform.openai.com
2. 右上角头像 → API Keys → Create new secret key
3. 复制 sk-xxx 开头的密钥

**方案B - 通义千问（国内，推荐）**
1. 访问 https://dashscope.aliyun.com
2. 注册阿里云账号 → 开通 DashScope
3. 控制台 → API-KEY管理 → 创建API-KEY
4. 复制密钥

> 💡 **建议**：先用通义千问，有免费额度，无需翻墙

### 2.3 微信支付（暂时可跳过）

MVP 阶段可先不配置支付，会员购买功能会提示但不崩溃。

## 第三步：配置后端环境变量

```bash
# 进入后端目录
cd bg

# 复制配置模板
copy .env.example .env
```

用记事本或 VS Code 打开 `bg/.env`，按以下说明填写：

```env
# ========== 服务器 ==========
PORT=3000
NODE_ENV=development

# ========== 数据库（按你的MySQL配置填写）==========
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的MySQL密码           # ← 必填
DB_NAME=ai_copywriter

# ========== Redis（默认本地无密码）==========
REDIS_HOST=localhost
REDIS_PORT=6379
REDIS_PASSWORD=                      # 本地Redis通常留空

# ========== JWT（随机字符串，越长越安全）==========
JWT_SECRET=请替换成一个随机长字符串abc123xyz789   # ← 必填

# ========== 微信小程序 ==========
WECHAT_APPID=wx你的AppID             # ← 必填（第二步获取）
WECHAT_SECRET=你的AppSecret          # ← 必填（第二步获取）

# ========== 微信支付（MVP可先留空）==========
WECHAT_MCHID=
WECHAT_PAY_KEY=

# ========== AI 模型（二选一填写即可）==========
# 方案A - OpenAI
OPENAI_API_KEY=sk-你的OpenAI密钥     # 填了就用OpenAI
OPENAI_MODEL=gpt-3.5-turbo
OPENAI_BASE_URL=https://api.openai.com/v1

# 方案B - 通义千问
QWEN_API_KEY=你的通义千问密钥         # 填了就用千问
QWEN_MODEL=qwen-turbo

# ========== 其他（保持默认即可）==========
CONTENT_AUDIT_ENABLED=false
RATE_LIMIT_WINDOW_MS=60000
RATE_LIMIT_MAX=100
LOG_LEVEL=info
```

## 第四步：初始化数据库

```bash
# 1. 登录 MySQL
mysql -u root -p

# 2. 执行初始化脚本（在MySQL命令行中）
source C:/Users/PC/Desktop/微信小程序/bg/src/database/init.sql

# 3. 验证
show tables;
# 应看到：users, members, contents, collections, orders 等表

# 4. 退出
exit
```

---

## 第五步：启动后端服务

```bash
# 进入后端目录
cd bg

# 安装依赖（首次运行）
npm install

# 开发模式启动
npm run dev
```

看到以下输出表示成功：
```
[info] Server running on port 3000
[info] MySQL connected successfully
[info] Redis connected
```

**验证接口**：浏览器访问 http://localhost:3000/health，应返回：
```json
{"status": "ok", "timestamp": "..."}
```

---

## 第六步：配置并启动小程序

1. 打开**微信开发者工具**
2. 新建项目：
   - 目录选择：`C:\Users\PC\Desktop\微信小程序\wx`
   - AppID：填写你的 AppID（或点「测试号」用测试AppID）
   - 框架：**不使用云服务**
3. 打开 `wx/app.js`，确认 `baseUrl`：
   ```js
   baseUrl: 'http://localhost:3000/api'  // 本地开发时保持不变
   ```
4. 点击工具栏「编译」，小程序启动！

## 常见问题排查

### ❌ MySQL 连接失败
```
Error: Access denied for user 'root'@'localhost'
```
→ 检查 `.env` 中的 `DB_PASSWORD` 是否正确

### ❌ Redis 连接失败
```
Error: connect ECONNREFUSED 127.0.0.1:6379
```
→ Redis 未启动，运行 `redis-server`（Windows 在安装目录找到 `redis-server.exe` 双击）

### ❌ AI 生成失败
```
AI生成失败，请稍后重试
```
→ 检查 `OPENAI_API_KEY` 或 `QWEN_API_KEY` 是否填写正确，网络是否可以访问 AI 服务

### ❌ 小程序登录失败
```
登录失败，请重试
```
→ 检查 `WECHAT_APPID` 和 `WECHAT_SECRET` 是否与微信后台一致

### ❌ 微信开发者工具提示「不合法域名」
→ 开发阶段正常现象，在微信开发者工具：「详情」→「本地设置」→ 勾选「**不校验合法域名**」
