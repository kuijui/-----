# 🚀 AI小红书文案生成器 - 快速启动指南

## 📋 前置准备

### 必须安装的软件

1. **Node.js** (v18+)
   - 下载：https://nodejs.org/
   - 验证：`node -v`

2. **MySQL** (v8.0+)
   - Windows: https://dev.mysql.com/downloads/mysql/
   - 验证：`mysql --version`

3. **Redis** (v7.0+)
   - Windows: https://github.com/tporadowski/redis/releases
   - 验证：`redis-cli --version`

4. **微信开发者工具**
   - 下载：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 必须申请的账号

1. **微信小程序**
   - 注册：https://mp.weixin.qq.com/
   - 获取：AppID 和 AppSecret

2. **OpenAI API** 或 **通义千问**
   - OpenAI: https://platform.openai.com/api-keys
   - 通义千问: https://dashscope.console.aliyun.com/

## 🔧 后端启动步骤

### 1. 安装依赖

```bash
cd bg
npm install
```

### 2. 配置环境变量

复制 `.env.example` 为 `.env`：

```bash
cp .env.example .env
```

编辑 `.env` 文件，填写以下**必填**配置：

```env
# 数据库配置（必填）
DB_PASSWORD=你的MySQL密码

# 微信小程序配置（必填）
WECHAT_APPID=你的小程序AppID
WECHAT_SECRET=你的小程序Secret

# AI配置（二选一）
# 方案1：使用OpenAI
OPENAI_API_KEY=sk-your-openai-api-key

# 方案2：使用通义千问
# QWEN_API_KEY=your-qwen-api-key

# JWT密钥（必填，建议随机生成）
JWT_SECRET=your_random_secret_key_here
```

### 3. 初始化数据库

```bash
# 登录MySQL
mysql -u root -p

# 执行初始化脚本
source src/database/init.sql

# 或者直接执行
mysql -u root -p < src/database/init.sql
```

### 4. 启动Redis

```bash
# Windows
redis-server

# 或者作为服务启动（Windows）
redis-server --service-start
```

### 5. 启动后端服务

```bash
# 开发模式（推荐）
npm run dev

# 或生产模式
npm start
```

看到以下信息表示启动成功：
```
服务器启动成功，端口：3000
MySQL数据库连接成功
Redis连接成功
```

### 6. 测试后端

访问：http://localhost:3000/api/health

应该返回：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "status": "healthy",
    "timestamp": "...",
    "version": "1.0.0"
  }
}
```

## 📱 小程序启动步骤

### 1. 打开微信开发者工具

- 选择"导入项目"
- 项目目录选择：`wx` 文件夹
- AppID：填写你的小程序AppID（或使用测试号）

### 2. 配置后端地址

编辑 `wx/app.js`，修改 `baseUrl`：

```javascript
globalData: {
  baseUrl: 'http://localhost:3000/api'  // 本地开发
  // baseUrl: 'https://your-domain.com/api'  // 正式环境
}
```

### 3. 开发者工具设置

- 点击右上角"详情"
- 勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"
- 勾选"启用调试"

### 4. 编译运行

点击"编译"按钮，即可在模拟器中看到小程序

### 5. 真机调试

- 点击"预览"按钮
- 使用微信扫码
- 在手机上测试

## 🐛 常见问题排查

### 后端问题

**问题1：MySQL连接失败**
```
解决：
1. 检查MySQL服务是否启动
2. 检查.env中的DB_PASSWORD是否正确
3. 检查数据库ai_copywriter是否已创建
```

**问题2：Redis连接失败**
```
解决：
1. 检查Redis服务是否启动
2. Windows: redis-server --service-start
3. 检查端口6379是否被占用
```

**问题3：AI生成失败**
```
解决：
1. 检查OPENAI_API_KEY是否正确
2. 检查网络是否能访问OpenAI
3. 检查API额度是否充足
4. 可以切换到通义千问
```

### 小程序问题

**问题1：网络请求失败**
```
解决：
1. 检查后端服务是否启动（访问http://localhost:3000/api/health）
2. 检查app.js中的baseUrl是否正确
3. 检查开发者工具是否勾选"不校验合法域名"
```

**问题2：登录失败**
```
解决：
1. 检查WECHAT_APPID和WECHAT_SECRET是否正确
2. 检查AppID是否与小程序项目匹配
3. 查看后端日志错误信息
```

**问题3：页面空白**
```
解决：
1. 打开调试器查看错误信息
2. 检查pages是否在app.json中注册
3. 检查代码是否有语法错误
```
