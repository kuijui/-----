### ⏳ 阶段6：凭证接入与真机联调（预计 30 分钟）

**需要你手动完成**，打开 `bg/.env` 填写：

```env
# 必填 ①：微信小程序凭证（mp.weixin.qq.com 获取）
WECHAT_APPID=wx你的真实AppID
WECHAT_SECRET=你的真实AppSecret

# 必填 ②：AI Key（二选一）
QWEN_API_KEY=你的通义千问Key      # 推荐，国内可用，有免费额度
# OPENAI_API_KEY=sk-你的OpenAI Key
```

填完后：
```bash
# 重启后端
cd bg && npm run dev

# 微信开发者工具打开
# 目录：wx/
# AppID：同上面填写的
```

**预期结果**：可以完整走通「登录 → 生成文案 → 查看结果 → 收藏 → 历史记录」流程。
