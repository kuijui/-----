# AI小红书文案生成器 - 微信小程序

## 📋 项目概述

一款基于AI技术的小红书文案生成工具，帮助用户快速生成高质量、吸引人的小红书内容，包括标题、正文、话题标签等。

## 🎯 项目定位

**目标用户**：小红书博主、电商运营、内容创作者、营销人员

**核心价值**：
- 3秒生成爆款文案，节省创作时间
- AI学习小红书爆款逻辑，提高内容质量
- 多种文案风格，适配不同场景
- 一键复制，无缝对接小红书发布
